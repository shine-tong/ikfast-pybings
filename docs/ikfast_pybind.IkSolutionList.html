ikfast_pybind.IkSolutionList¶

class ikfast_pybind. IkSolutionList

多个逆运动学解的容器，支持 Python 迭代。

此类包含 IKFast 求解器返回的所有逆运动学解。它支持 Python 的标准容器操作，如索引、迭代和长度查询。

方法

__len__()

返回解的数量

__getitem__(index)

通过索引访问解

示例

import ikfast_pybind as ik
import numpy as np

translation = np.array([0.5, 0.0, 0.5])
rotation = np.eye(3).flatten()

# 获取解列表
solution_list = ik._ikfast_pybind.compute_ik_raw(translation, rotation, None)

# 获取解的数量
print(f"找到 {len(solution_list)} 个解")

# 通过索引访问
if len(solution_list) > 0:
    first_solution = solution_list[0]
    joints = first_solution.get_solution()
    print(f"第一个解: {joints}")

# 迭代所有解
for i in range(len(solution_list)):
    solution = solution_list[i]
    joints = solution.get_solution()
    print(f"解 {i+1}: {joints}")
